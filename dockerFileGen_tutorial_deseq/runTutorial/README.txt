Files from tutorial http://bioconductor.org/packages/devel/bioc/vignettes/DESeq2/inst/doc/DESeq2.html
Download script.sh (linux/MACOSX) or script.cmd. 
Execute the script. The docker image will be downloaded from repbioinfo repository
At the end of the execution of the script the docker container is run and the jupyter lab can be accessed at localhost:8888
Using the jupyter GUI, locate the /sharedFolder/deseq_Tutorial.ipynb. Execute it and check how easy can be to share an algorithm/analysis among colleagues or reviewer.
