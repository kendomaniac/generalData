Exammple of the use of CREDOgui as tool to share a set of analyses. The example is taken from the tutorial of DESeq2 (http://bioconductor.org/packages/devel/bioc/vignettes/DESeq2/inst/doc/DESeq2.html).
Quick execution: download in a local folder all the files in runDocker folder. Execute the script.sh (linux/MACOSX) or script.rmd (Window). The image will be downloaded from docker.io repository.
Compiling: as alternative option you can clone the present repository and execute the script.sh/cmd in the Dockerfile folder. This will compile from scratch the docker image.
